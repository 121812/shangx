## 一键安装
```bash
curl https://raw.githubusercontent.com/121812/shangx/main/install.sh | bash
```

## 手动安装
```bash
git clone https://github.com/121812/shangx.git
chmod 744 shangx/shangx.py
mv shangx/shangx.py /sbin/shangx
```

## 运行
```bash
shangx
```
